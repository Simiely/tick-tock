# 🕐 Tick-Tock

> Bong~ Bong~ 到点了！

一个纯 HTML 的整点语音报时小工具。无需安装任何依赖，浏览器打开即用。

## 亮点

- **🗣️ 语音报时** — 到整点/半点，自动播报"现在时间，上午八点整"
- **双引擎** — 系统语音（离线）+ 在线 TTS（Google），兼容 Android
- **4 种传统钟声** — 大本钟、门铃、旋律、柔和，各取所好
- **免打扰时段** — 深夜不吵你（默认 22:00~07:00）
- **滚轮选择器** — 设置时间像 iOS 一样上下滑
- **报时日志** — 今日报了哪些时间一目了然
- **离线可用** — 音效用 Web Audio 合成，不需要任何外部文件

## 快速开始

```bash
# 直接打开
open 整点报时.html

# 或起个本地服务器
python3 -m http.server 8080
```

然后浏览器打开，**点一下页面**激活语音引擎即可。
