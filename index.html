🚀 立即使用 📦 GitHub

✨ 功能特性

一个 HTML 文件，搞定整点报时，浏览器打开即用。

🗣️

语音报时

到整点或半点，自动语音播报"现在时间，上午八点整"，系统语音与在线 TTS 双引擎。

🔔

4 种钟声

大本钟、门铃、旋律、柔和——传统音效用 Web Audio 合成，不需要任何外部音频文件。

🔕

免打扰时段

设置深夜静音区间，滚轮选择器像 iOS 一样上下滑动，优雅设置。

📋

报时日志

自动记录今日所有报时，设置持久化到本地存储，刷新不丢失。

📱

移动端适配

响应式设计，支持振动反馈，为后续打包 APK 做好准备。

🌐

多源兜底

系统语音 → Google TTS → Edge TTS → 音乐报时，四级 fallback，网络断了也能出声。

🎵

音乐报时

终极离线方案：用旋律编码时间——每小时的音高都不同，永远不会哑火。

🎯 如何使用

三步走，开箱即用。

1

下载文件

从 GitHub 下载 整点报时.html，或直接 点此打开 在线版。

2

点击激活

在页面上**随意点击一下**，激活语音引擎和音频上下文。

3

等着报时

等到整点，它就会开口告诉你现在几点。你也可以点"测试报时"先听个效果。
